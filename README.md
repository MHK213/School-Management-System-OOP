# School Management System (C++ OOP)

This is a **console-based School Management System** built using **C++**.

The system provides full management for:

- 🎓 Students
- 👨‍🏫 Teachers
- 📚 Courses
- 📝 Enrollments
- 📊 Reports & Statistics

The project is designed using a **layered architecture** to ensure clean code, scalability, and separation of concerns
